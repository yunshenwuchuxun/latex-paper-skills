# Results folder

This folder is the interface for `results-backfill`.

The CSV/JSON files retained here were generated from verified runs of the empirical pipeline against a local download of ResOpsUS v2 during project development. The raw dataset has been removed from the public repository, but these result files and the derived figures in `../figures/` are kept as evidence that the workflow completed real experiments.

Place verified experiment outputs here (CSV/JSON). Recommended canonical files:
- main_results.csv
- ablation_results.csv
- robustness_results.csv
- efficiency_results.csv

Do not fabricate numbers. Use placeholders in the paper until verified results exist.
